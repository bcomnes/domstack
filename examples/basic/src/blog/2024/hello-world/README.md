# Hello World

Welcome to the first post on this example blog. This is a simple markdown page rendered with the `blog` layout.

## What is DOMStack?

DOMStack is a static site generator built on Node.js and esbuild. It lets you build websites with markdown, HTML, and JavaScript pages — all without a heavy framework.

Some things it supports out of the box:

- Markdown pages with frontmatter
- JavaScript pages with full async rendering
- HTML pages with Handlebars templating
- Per-page CSS and client JS bundles
- Global layouts
- RSS/Atom feeds via template files
